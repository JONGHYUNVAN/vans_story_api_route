    Class S3UploadError

    S3 업로드 관련 에러를 나타내는 클래스입니다. AWS S3에 파일을 업로드하는 과정에서 발생하는 에러를 처리합니다.

    Example: Request

    {
      "file": "<업로드할 이미지 파일>"
    }

    Example: Response

    {
      "name": "S3UploadError",
      "message": "AWS 인증에 실패했습니다.",
      "code": "AUTH_FAILED"
    }

    주요 에러 상황:

    • AWS 인증 실패 (AUTH_FAILED)
    • 네트워크 오류 (NETWORK_ERROR)
    • 권한 부족 (PERMISSION_DENIED)
    • 버킷 접근 실패 (BUCKET_ACCESS_FAILED)
    • 환경 변수 누락 (ENV_MISSING)

    Example: AWS 인증 실패

    {
      "name": "S3UploadError",
      "message": "AWS 인증에 실패했습니다.",
      "code": "AUTH_FAILED"
    }

    Example: 네트워크 오류

    {
      "name": "S3UploadError",
      "message": "네트워크 오류가 발생했습니다.",
      "code": "NETWORK_ERROR"
    }

    Example: 권한 부족

    {
      "name": "S3UploadError",
      "message": "버킷 접근 권한이 없습니다.",
      "code": "PERMISSION_DENIED"
    }

    Example: 환경 변수 누락

    {
      "name": "S3UploadError",
      "message": "환경 변수 누락",
      "code": "ENV_MISSING"
    }
