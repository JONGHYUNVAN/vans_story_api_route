    Interface S3UploadOptions

    S3 업로드 옵션 인터페이스

    S3에 파일을 업로드할 때 사용되는 설정 옵션을 정의합니다.

    Example: 기본 옵션

    {
      "contentType": "image/webp",
      "publicRead": true,
      "metadata": {
        "originalName": "example.jpg"
      },
      "prefix": "images/"
    }
