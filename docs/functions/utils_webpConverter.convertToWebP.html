    Function convertToWebP

    • convertToWebP(
          buffer: ArrayBuffer | Buffer<ArrayBufferLike>,
          options?: WebPConversionOptions,
      ): Promise<Buffer<ArrayBufferLike>>

      이미지를 WebP 형식으로 변환합니다.

      Parameters

      • buffer: ArrayBuffer | Buffer<ArrayBufferLike>

        변환할 이미지 데이터 (Buffer 또는 ArrayBuffer)

      • options: WebPConversionOptions = {}

        변환 옵션 (현재는 사용되지 않음)

      Returns Promise<Buffer<ArrayBufferLike>>

      WebP 형식으로 변환된 이미지 버퍼

      Throws

      이미지 변환 중 오류 발생 시

      지원하는 입력 형식:

      • JPEG, PNG, GIF, TIFF, WebP, AVIF 등 (Sharp가 지원하는 모든 형식)

      변환 특징:

      • 원본 크기 유지 (리사이징 없음)
      • 품질: 80% 고정
      • 메타데이터 보존
      • 압축 최적화

      Example: Request

      {
        "file": "<업로드할 이미지 파일>"
      }

      Example: Response (성공)

      "<Buffer 52 49 46 46 ... >"

      Example: Response (실패)

      {
        "name": "ImageProcessingError",
        "message": "지원하지 않는 이미지 형식입니다.",
        "code": "UNSUPPORTED_FORMAT"
      }

      Since

      1.0.0

      See

      ImageProcessingError - 이미지 처리 에러 클래스
