    Function uploadToS3

    • uploadToS3(fileBuffer: Buffer, options?: S3UploadOptions): Promise<string>

      이미지 파일을 AWS S3에 업로드합니다.

      이 함수는 이미지 파일을 AWS S3 버킷에 업로드하고, 업로드된 파일의 URL을 반환합니다.

      Parameters

      • fileBuffer: Buffer

        업로드할 이미지 파일 버퍼

      • options: S3UploadOptions = {}

        업로드 옵션

      Returns Promise<string>

      업로드된 파일의 URL (HTTPS)

      Throws

      업로드 실패 시

      업로드 과정:

      1. 환경 변수 검증
      2. 고유한 파일명 생성
      3. S3 클라이언트 설정
      4. 파일 업로드 실행
      5. 업로드된 파일 URL 반환

      파일명 생성 규칙:

      • 접두사 (옵션): S3UploadOptions.prefix
      • 타임스탬프: YYYYMMDD_HHmmss
      • 랜덤 문자열: 8자리 16진수
      • 확장자: .webp

      예시: 'images/20240315_143022_a1b2c3d4.webp'

      에러 처리:

      • 환경 변수 누락
      • AWS 인증 실패
      • 네트워크 오류
      • 권한 부족
      • 버킷 접근 실패

      Example: Request

      {
        "file": "<업로드할 이미지 파일>"
      }

      Example: Response (성공)

      "https://my-bucket.s3.ap-northeast-2.amazonaws.com/images/20240315_143022_a1b2c3d4.webp"

      Example: Response (실패)

      {
        "name": "S3UploadError",
        "message": "AWS 인증에 실패했습니다.",
        "code": "AUTH_FAILED"
      }

      Since

      1.0.0
